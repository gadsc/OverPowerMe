OverPowerMe
===========

Projeto open source para simulações de runas/builds do League of Legends.

Utilizaremos as seguintes tecnologias:

Desenvolvimento aplicação

Front-End:
- VRaptor
- Html 5
- CSS 3
- JQuery

Back-End:
- CDI
- Spring
- JPA
- Cache
- Spring Data
- Spring Security

Infraestrutra
Server WEB:
- Glassfish 4.1
 
Barramento:
- WSO2

Application Server:
- Glassfish 4.1

Banco de Dados:
- MongoDB
